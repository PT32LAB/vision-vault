The note covers the eight dimensions, the data sources (Hansen/GFW, WRI Aqueduct, UCDP, OSM, public climate/soil/solar sets), QGIS/Jupyter export, and the "filter, never score/rank" philosophy. **Key honest caveat surfaced and emphasised:** the live tool covers Europe + North America only (20 regions), so it is NOT usable as-is for Coroico/Bolivia — its value to BARK is as a *methodology template* to re-run with regional datasets.
