# Knowledge Cosmos

Every piece of BARK's knowledge exists in a multidimensional space. Each node is scored across six dimensions (0.0-1.0) from Oleg's NetworkTribes framework.

![The famous Death Road winding through the Yungas mountains](/images/yungas/death-road.jpg)

## The Six Dimensions

| Dimension | Description |
|-----------|-------------|
| **Autonomy** | Independence from state/corporate systems |
| **Technology** | Technical infrastructure required |
| **Governance** | Decision-making relevance |
| **Economic** | Economic self-sufficiency contribution |
| **Resilience** | Crisis survival improvement |
| **Scalability** | Works at larger scale |
